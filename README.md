# 🎮 GameLib

```
C++ 遊戲框架，支援 MVC 架構、狀態機、Console View、多執行緒與 Server/Client 模組。
提供一套輕量、可擴充的基礎架構，讓 C++ 遊戲開發變得更簡單、更有組織，適合用於：
```

- 回合制或即時制遊戲原型開發
- 學習設計模式（如 MVC、State Machine）
- 終端機互動遊戲系統(提供輕量 ConsoleView 實作)
- 整合你自己的圖形 / 音效 / 網路等外部 Lib

🚀 **支援擴充性強、模組分離清晰，易於維護與多人協作！**
